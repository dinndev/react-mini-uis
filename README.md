# react-mini-uis

# What is this ?

Light easily applied ui!

## Install

```bash
npm install --save react-mini-uis
```

## Usage

```jsx
import { Button } from 'react-mini-uis'

class Example extends Component {
  render() {
    return <Button theme="Primary" size="Small"> Sign Up <Button>
  }
}
```
